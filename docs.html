Myco Language Documentation

Introduction

Myco is a lightweight scripting language designed for simplicity and readability. It combines the performance of C with the elegance of Lua, creating a language that's both powerful and easy to learn.

This documentation covers all aspects of the Myco language, from basic syntax to advanced features.

Installation

To get started with Myco, you'll need to build it from source:

# Clone the repository
git clone https://github.com/IvyMycelia/Myco.git
cd Myco

# Build the project
make

Basic Syntax

Myco uses a clean, readable syntax inspired by Lua and C:

# This is a comment
let message = "Hello, Myco!";
print(message);

Variables

Variables in Myco are declared using the let keyword:

let name = "Alice";
let age = 25;
let is_active = 1;  # 1 = true, 0 = false

Functions

Functions are defined using the func keyword:

# Traditional syntax
func greet(person):
    print("Hello,", person);
end

# Arrow syntax for return types
func add(a: int, b: int) -> int:
    return a + b;
end

Control Flow

Myco supports standard control flow constructs:

# If statements
if age >= 18:
    print("Adult");
else:
    print("Minor");
end

# Loops
for i = 1 to 10:
    print(i);
end

Data Structures

Myco provides built-in support for lists and maps:

# Lists
let numbers = [1, 2, 3, 4, 5];

# Maps/Objects
let person = {
    name: "Alice",
    age: 25,
    city: "Forest"
};

Error Handling

Use try-catch blocks for error handling:

try:
    let result = divide(10, 0);
catch error:
    print("Error:", error);
end

Standard Library

Myco includes a comprehensive standard library:

# Math functions
let result = pow(2, 8);
let sqrt_result = sqrt(16);

# Utility functions
debug(variable);
let type_info = type(variable);

Complete Examples

Here's a complete Myco program:

# Simple calculator
func calculator():
    print("Myco Calculator");
    
    let numbers = [10, 5, 3];
    
    for num in numbers:
        let square = pow(num, 2);
        print(num, "squared is", square);
    end;
end

calculator();
