Quick Start Guide

Follow these steps to get Myco running on your system quickly.

Prerequisites

Before building Myco, ensure you have the following tools installed:

  • GCC compiler (version 8.0 or later) or Clang
  • Make build system
  • Git for cloning the repository

Build Steps

Follow these commands to build Myco from source:

Terminal Commands
# Clone the repository
git clone https://github.com/IvyMycelia/Myco.git
cd Myco

# Navigate to source directory
cd myco

# Build the project
make

# Test the installation
./myco --version

Platform-Specific Setup

Different platforms may require additional setup steps:

Linux

Most distributions should work out of the box. For Ubuntu/Debian systems:

sudo apt update
sudo apt install build-essential git

macOS

Install Xcode Command Line Tools for development dependencies:

xcode-select --install

Windows

Use WSL2 (Windows Subsystem for Linux) for the best experience:

wsl --install -d Ubuntu

After installing WSL2, follow the Linux instructions above.
