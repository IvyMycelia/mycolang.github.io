Myco Language Grammar

Grammar Overview

The Myco language follows a context-free grammar defined using Backus-Naur Form (BNF) notation. This grammar defines the syntax rules for valid Myco programs.

Version: v1.4.0 - Float Support & Library Import System

Last Updated: December 2024

Status: Complete and Current

Myco's grammar is designed to be simple yet expressive, with clear precedence rules and intuitive syntax.

Complete BNF Grammar

Program Structure

Program Structure 
# Complete program
<program> ::= <statement_list>

<statement_list> ::= <statement>*
<statement> ::= <expression_statement> | <declaration_statement> | 
                <control_statement> | <function_declaration> | 
                <return_statement> | <break_statement> | 
                <continue_statement> | <library_import_statement>

<block> ::= "end" | <statement_list> "end"

Lexical Elements

Lexical Elements 
# Identifiers
<identifier> ::= <letter> (<letter> | <digit> | "_")*
<letter> ::= "a" | "b" | "c" | ... | "z" | "A" | "B" | ... | "Z"
<digit> ::= "0" | "1" | "2" | ... | "9"

# Literals
<literal> ::= <number_literal> | <string_literal> | 
              <boolean_literal> | <array_literal> | <object_literal>

<number_literal> ::= <integer> | <float>
<integer> ::= <digit>+ | "-" <digit>+
<float> ::= <digit>+ "." <digit>* | "." <digit>+ | 
            "-" <digit>+ "." <digit>* | "-" "." <digit>+

<string_literal> ::= '"' <string_content>* '"'
<boolean_literal> ::= "0" | "1" | "True" | "False"

<array_literal> ::= "[" <expression_list> "]"
<object_literal> ::= "{" <property_list> "}"
