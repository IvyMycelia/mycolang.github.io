Myco Programming Language Examples

See how expressive and readable Myco code can be. These examples demonstrate the language's versatility and power.

Hello World

Basic Output 
print("Hello, Myco World!");

Simple and clean - Myco makes basic programming tasks straightforward and intuitive.

Variables and Types

Dynamic Typing 
let name = "Alice";
let age = 25;
let isActive = True;
let scores = [95, 87, 92];

Myco's dynamic typing system automatically handles different data types with clear, readable syntax.

Functions

Function Definition 
func greet(name):
    return "Hello, " + name + "!";
end

let message = greet("World");

Define functions with clear syntax and optional type annotations for better code clarity.

Control Flow

Conditionals and Loops 
for i in [1, 2, 3]:
    if i % 2 == 0:
        print(i, "is even");
    end
end

Express complex logic with clean, readable control flow statements that are easy to understand and maintain.

Data Structures

Arrays and Objects 
let users = [
    {"name": "Alice", "age": 25},
    {"name": "Bob", "age": 30}
];

for user in users:
    print(user."name", "is", user."age");
end

Work with complex data structures using intuitive syntax that makes data manipulation straightforward and efficient.

Error Handling

Try-Catch Blocks 
try:
    let result = 10 / 0;
catch error:
    print("Error occurred:", error);
end

Handle errors gracefully with built-in try-catch blocks that make your code more robust and reliable.

About Myco Programming Language

Created by Ivy Mycelia

Myco is the brainchild of Ivy Mycelia, a passionate developer and programming language enthusiast. The language was designed with a clear vision: to create a programming language that combines the elegance of modern syntax with the power and versatility that developers need in today's fast-paced software development environment.

Design Philosophy

Myco's design philosophy centers around expressiveness, readability, and developer productivity. We believe that programming should be intuitive and enjoyable, not a struggle with complex syntax or arcane language rules.

Key Design Principles

  • Simplicity: Clean, intuitive syntax that's easy to learn and use
  • Expressiveness: Write clear, readable code that expresses intent
  • Versatility: Handle diverse programming tasks with a single language
  • Modernity: Incorporate contemporary programming language features and best practices
  • Community: Open development process with community input and feedback

Current Status

Myco is currently in active development with version 1.4.0 featuring float support, library import systems, and file I/O capabilities. The language continues to evolve based on developer feedback and emerging programming language research.
