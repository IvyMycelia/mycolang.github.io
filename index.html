Myco Language

A lightweight scripting language designed for simplicity and readability

Think C meets Lua. Myco combines the performance of C with the elegance of Lua, creating a language that's both powerful and easy to learn.

View on GitHub Read Documentation
example.myco 
func fibonacci(n):
    if n <= 1:
        return n;
    end
    return fibonacci(n - 1) + fibonacci(n - 2);
end

let result = fibonacci(10);
print("Result:", result);

Functions

func greet(person):
    return "Hello, " + person;
end

let message = greet("World");
print(message);

Control Flow

let x = 10;

if x > 5:
    print("x is greater than 5");
else:
    print("x is 5 or less");
end

for i = 1 to x:
    print(i);
end
