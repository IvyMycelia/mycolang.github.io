Myco Programming Language

High-Level, Expressive & Versatile Programming Language

Myco is a modern, high-level programming language created by Ivy Mycelia that combines expressive syntax with powerful features. Perfect for both beginners and experienced developers, featuring comprehensive libraries, dynamic typing, and cross-platform compatibility.

View on GitHub Read Documentation
example.myco 
func fibonacci(n):
    if n <= 1:
        return n;
    end
    return fibonacci(n - 1) + fibonacci(n - 2);
end

let result = fibonacci(10);
print("Result:", result);

Myco Programming Language Features

Discover what makes Myco a powerful, high-level programming language perfect for modern development

Clean Syntax

Intuitive syntax that's easy to read and write. No unnecessary complexity.

Syntax Features

  • Minimal keywords
  • Clear structure
  • Readable formatting
  • Consistent patterns

Dynamic Typing

Flexible type system with automatic type inference. Variables adapt to your needs.

Type System

  • Automatic inference
  • Runtime type checking
  • Flexible assignments
  • Type conversion

Control Flow

Standard control structures: if/else, loops, functions, and error handling.

Control Structures

  • If/else statements
  • For and while loops
  • Function definitions
  • Error handling

Data Structures

Built-in support for lists, maps, and custom data types.

Built-in Types

  • Dynamic arrays
  • Hash maps
  • Custom structs
  • Type safety

Standard Library

Comprehensive library with math functions, utilities, and I/O operations.

Library Features

  • Math operations
  • File I/O
  • String utilities
  • System functions

Cross Platform

Runs on Windows, macOS, and Linux. Write once, run anywhere.

Platform Support

  • Windows 10+
  • macOS 10.14+
  • Linux (glibc 2.17+)
  • ARM64 support

Myco Programming Language Examples

See Myco in action with practical code examples showcasing the language's expressive syntax and powerful features

Variables and Types

let name = "Alice";
let age = 25;
let active = 1;

print("Name:", name);

Functions

func greet(person):
    return "Hello, " + person;
end

let message = greet("World");
print(message);

Control Flow

let x = 10;

if x > 5:
    print("x is greater than 5");
else:
    print("x is 5 or less");
end

for i = 1 to x:
    print(i);
end

Data Structures

# Lists
let numbers = [1, 2, 3, 4];

# Maps
let user = {
    "name": "Bob",
    "age": 30
};

print(user["name"]);
