Build from Source

Get the latest features by building Myco from source code

Prerequisites

Required Tools

  • GCC compiler (version 8.0 or later)
  • Make build system
  • Git (for cloning repository)

Optional Tools

  • CMake (alternative build system)
  • Clang compiler (alternative to GCC)
  • Valgrind (for memory debugging)

Build Steps

Clone and Build 
# Clone the repository
git clone https://github.com/IvyMycelia/Myco.git
cd Myco

# Navigate to the source directory
cd myco

# Build the project
make

# Install (optional, requires sudo on Linux/macOS)
sudo make install

# Test the installation
./myco --version

Platform-Specific Notes

Linux

Most distributions should work out of the box. Ensure you have the development tools installed.

Ubuntu/Debian 
sudo apt update
sudo apt install build-essential git

macOS

Install Xcode Command Line Tools or use Homebrew for development dependencies.

macOS Setup 
# Install Xcode Command Line Tools
xcode-select --install

# Or use Homebrew
brew install gcc make

Windows

Use WSL2, MinGW, or Visual Studio for building. WSL2 is recommended for the best experience.

WSL2 Setup 
# Install WSL2 and Ubuntu
wsl --install -d Ubuntu

# Then follow Linux instructions
